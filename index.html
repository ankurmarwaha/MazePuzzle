How to play

  • Move: Arrow keys, WASD, or swipe on the maze.
  • Goal: Reach the green exit square.
  • New maze: Changes the layout but keeps the size.
Hosting

This is a static site. Upload the folder to any web server, or run a local static server. 

cd maze-puzzle
python -m http.server 8000

Then open http://localhost:8000.

Tip: press R to reset, N for new maze, H to toggle solution.
